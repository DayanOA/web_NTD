Modelo Vista Controlador


Es un patron que pertence al la familia componente-conector, separa la lógica de negocio de la interfaz de usuario. Facilita la evolución por separado de ambos aspectos, incrementando de esta forma la reutilización y flexibilidad.




Servidores


Es el componente fisico o virtual capaz de realizar el desplique de la aplicación, es donde se encuentra la información de la base de datos y las vistas a las cuales el usuario queire aceder por peticiones.




Arquitectura JPA


Es una API que proporciona un mecanismo para gestionar la persistencia y la correlación y funciones relacionales de objetos.




Controlador


El controlador se encarga de realizar la unión de la información con las vista de esta al usuario final, en este trancurso de solicitudes, veremos el CRUD "crear leer actualizar y borrar", peticiones entre otras cosas mas como la utilización de frameworks.




Vista


Empieza la parte de diseño y desarrollo front-end, todo lo que se ejecuta en la computadora del usuario fuera del servidor, aprenderemos el etiquetado de HTML5 y con esto podras utilizar preprocesadores como HAML JADE entre otros. para dar estilos CSS aprenderemos el uso de preprocesadores como SASS, y la iteracción con el usuaro con el uso de JavaScript.




Django


Es un potente framework de desarrollo Web con el que podemos utilzar el leguanje Phyton para el desarrollo de aplicaciones, agregando velocidad al desarrollo y seguridad.




Node JS


¿ JavaScript desde el servidor?, Node JS es JavaScript orientado a eventos asíncronos, esta diseñado para construir aplicaciones en red escalables.
